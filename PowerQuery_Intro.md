## PowerQuery
PowerQuery is used to do in depth data transformation. 
* Whenever we tranform Data in PowerBi, a new window appears on the screen, this is PowerQuery window. It is a separate window to perform PowerQuery operations. 
* To zoom in/out we use ```ctr + shift + "+" / "-"```. 
* We can see applied steps on right hand side of the screen. Where each step is mentioned in chronological order.
* We can filter the columns of table and this is also considered as a tranformation step which is reflected on the 'applied step' section. 
* In the table there are chances of null values, one way to deal with them is simply replace them. 
* By clicking on column name we can filter rows or change ascending to descenting order on the click. 
* Above the table the code written is called M-language. This is a basic language in PowerQuery which allows user to make changes using code. 
* "Add Column" option in the top helps us add new conditional column by giving different condition which are dependent on other columns. 

### Summary 
PowerQuery used to Transform data i.e. to make it ready for visulization.   
General tasks PowerQuery does: 
* Sort/Filter 
* Handle Null values
* Add columns 
* Change Datatypes
