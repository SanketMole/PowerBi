## PowerBi
A Visualization tool used to make Data more understandable to clients. 
PowerBi is made by Microsoft. 

* PowerBi has options to work in multiple layouts. 
* PowerBi provides an option to connect with multiple Data sources. 
* 'Report View' is the area to create visuals. 
* 'Table view' helps one check and modify the tables. 
* 'Mobile view' is used to show layout of charts in mobile.
